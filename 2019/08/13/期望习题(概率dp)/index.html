期望习题(概率dp)

Posted on 2019-08-13

几道例题

1.Collecting Bugs POJ-2096（期望DP）

题意：有n种bug，s个系统会出现bug，bug数量可以是无穷多的，有一个人一天能够找一个bug，bug出现在各个系统是等概率的，系统中发现的bug也是等概率的属于每种类别，问找到每种bug至少找到一个，每个系统至少找到一个bug的时间期望.

题解：由题意可知，一个bug出现在某个系统的概率是$\frac{1}{s}$，属于某个类型的概率是$\frac{1}{n}$，

设dp[i] [j]表示已经在i个系统中找到了 j种bug，距离全部找到（即达到目标状态）还需要天数的期望，所以有dp[s] [n]=0,本题采用逆推求期望，最终要求的即为dp[0] [0].

所以dp[i] [j]可由以下状态推得:

  1. dp[i] [j] —>旧系统，旧bug（即这个bug是在已经找到过bug的系统中再找到的，并且已经找到过了它同种类的bug已经被找到过了）

    $P_1=\frac{i}{s}*\frac{j}{n}$

  2. dp[i+1] [j] —>新系统，旧bug

    $P_2=\frac{\left( s-i \right)}{s}*\frac{j}{n}$

  3. dp[i] [j+1]—>旧系统，新bug

    $P_3=\frac{i}{s}*\frac{\left( n-j \right)}{n}$

  4. dp[i+1] [j+1]—>新系统，新bug

    $P_4=\frac{\left( s-i \right)}{s}*\frac{\left( n-j \right)}{n}$

所以有：dp[i] [j]=$P_1$ *dp[i] [j] +$P_2$ *dp[i+1] [j]+$P_3$ *dp[i] [j+1]+$P_4$ *dp[i+1 [j+1]+1

化简得：$dp\left[ i \right] \left[ j \right] =\frac{\left( s-i \right) jdp\left[ i+1 \right] \left[ j \right] +i*\left( n-j \right) *dp\left[ i \right] \left[ j+1 \right] +\left( s-i \right) \left( n-j \right) *dp\left[ i+1 \right] \left[ j+1 \right] +ns}{sn-ij}$

2.Race to 1 Again LightOJ - 1038 (期望DP)

题意：给出一个数D，在1~D中选一个它的因子，让D除以这个因子得到一个新的D，这个新的D再选一个因子继续除，直到D=1，求进行的除法次数的期望

题解：设dp[i]表示由i变成1的除法次数期望，易知dp[1]=0

设j是i的一个因子，dp[i]由dp[j]转移而来

$dp\left[ i \right] =\frac{\varSigma dp\left[ j \right]}{num}+1$（设i=k*j，k为选择的因子，1≤k≤i，num表示i的因子个数）

当k=1时，$dp\left[ i \right] =\frac{dp\left[ i \right] +\varSigma dp\left[ j \right]}{num}+1$（j<i)dp[i]

化简可以得到$dp\left[ i \right] =\frac{num+\varSigma dp\left[ j \right]}{num-1}$（j<i)

dp[2]=(2+dp[1])/1=2

3.Dice (III) LightOJ - 1248 (期望+几何分布)

题意：给你一个有n个面的骰子，问你这n面都出现过的投掷次数的期望

题解：**设dp[i]为扔出i个不同面后，到达目标n还需要的投掷次数的期望，**dp[n]=0;

每次投掷其实只有两种情况：出现过的$P_1=\frac{i}{n}$和 没出现过的$P_2=\frac{n-i}{n}$

所以有：$dp\left[ i \right] =\frac{i}{n}dp\left[ i \right] +\frac{n-i}{n}dp\left[ i+1 \right] +1$

化简得：$dp\left[ i \right] =dp\left[ i+1 \right] +\frac{n}{n-i}$

4.Activation HDU-4089 (概率DP+方程组)

题意：有n个人在排队注册游戏账号，对于每个用户，都可能遇到以下情况:

  1. 处理失败，该用户继续停留在原位等待重新注册，发生的概率为$P_1$
  2. 处理错误，该用户到队尾重新排队，发生的概率为$P_2$
  3. 处理成功，该用户出队，发生的概率为$P_3$
  4. 服务器故障，所有用户都不能注册了，发生的概率为$P_4$

Tomato现在排在第m个位置，求他排在第k位以前（包括第k位）时服务器故障的概率

题解：设dp[i] [j]表示队伍人数为i时，Tomato排在第j位的概率，我们最终要求的结果就是dp[n] [m]

状态转移方程：

  1. j==1时，**dp[i] [1]=$P_1$ dp[i] [1]+$P_2$ dp[i] [i]+$P_4$

    ( $P_3$对应的情况：如果Tomato成功，出队了就成了dp[i] [0]，而dp[i] [0]=0，就像如果$P_4$=0，即不存在服务器故障情况，输出的概率就为0了)

  2. 1<j≤k时，**dp[i] [j]=$P_1$ *dp[i] [j]+$P_2$ dp[i] [j-1]+$P_3$ dp[i-1] [j-1]+$P_4$

  3. j>k时，**dp[i] [j]=$P_1$ *dp[i] [j]+$P_2$ dp[i] [j-1]+$P_3$ dp[i-1] [j-1] (这里不能服务器故障，不然题目要求的事件就不会发生了)

要化简上面式子：

令$x_2=\frac{P_2}{1-P_1}$，$x_3=\frac{P_3}{1-P_1}$，$x_4=\frac{P_4}{1-P_1}$

如果我们对i从1->n递推的话，dp[i-1] [j-1]相对dp[i] [j-1]就可以看作是已经求解的常数项进行处理，

dp[i] [1]=$x_2$ * dp[i] [i]+c[1]， c[1]=$x_4$ (j==1时)

dp[i] [j]=$x_2$ * dp[i] [j-1]+c[j] ，c[j]=$x_3$ * dp[i-1] [j-1]+$x_4$ (j>k时不加$x_4$)

迭代求解dp[i] [i]:

求出dp[i] [i]后可根据dp[i] [1]=$x_2$ dp[i] [i]+c[1]，推出dp[i] [1],从而推出所有的dp[i] [j]*

注意：要特判p4=0的情况，题目的内存限制很小，32768 kB=$2^{25}$Byte，大概double型占8Byte，大概只能开$2^{22}$即大概 $4 * 10^6$ 大小的double型数组,而根据题目n * m,dp[n] [m]就已经 $4 * 10^6$，何况还有别的数组要占用内存，所以肯定是不够的，又因为只用到dp[i-1] []和dp[i] []的关系，所以用滚动数组进行处理
